LOGOS FOLDER
============

Place logo files in this directory.

Suggested files:
- logo.png          - Main logo (recommended: 512x512px)
- logo-small.png    - Small logo for navbar/favicon (recommended: 64x64px)
- logo-dark.png     - Logo variant for dark backgrounds
- logo-light.png    - Logo variant for light backgrounds
- favicon.ico       - Browser favicon (16x16, 32x32, 48x48)
- favicon.png       - PNG favicon for modern browsers

Supported formats: PNG, SVG, ICO, JPG
