ICONS FOLDER
============

Place icon assets in this directory.

Suggested files:
- faction-icons/    - Icons for each faction (Barada, etc.)
- trait-icons/      - Icons for different trait types
- ui-icons/         - General UI icons (if not using inline SVG)
- social/           - Social media icons

Recommended formats:
- SVG (preferred for scalability)
- PNG with multiple sizes (24x24, 32x32, 48x48)
